# nuxt-uid-module

[vue-uid](https://github.com/mya-ake/vue-uid) for Nuxt.js.

## Install

```bash
$ yarn add nuxt-uid-module
```

### nuxt.config.js

```JavaScript
module.exports = {
  // ...
  modules: [
    'nuxt-uid-module',
  ],
  // ...

  // options (See https://github.com/mya-ake/vue-uid#option)
  'uid-module': {
    name: 'uid'
  },
};
```
